### MAC주소, IP주소, Port번호가 식별하는 것
<br>

- MAC
  - **NIC**(Network Interface Card)를 식별하는 주소
  - 즉, **LAN 카드**를 말한다
  - NIC 당 1개
  - H/W주소지만 MAC 주소 변경 가능한가? **가능하다**
- IP
  - **Host**에 대한 식별자
  - Host: 인터넷에 연결된 컴퓨터
  - NIC 당 n개
- Port
  - 개발자: process 식별자
  - 네트워크 관리자: service 번호
  - 통신공사: interface 번호

  <br>

Network : 자원을 공유하기 위해 단말들이 서로 통신이 가능하도록 연결(LAN)
service : 네트워크에서 정의된 특정 기능 수행을 의미한다. 웹 서비스의 경우 http 서비스 식별자가 사용된다.
interface : LAN 케이블 단자는 컴퓨터가 속한 네트워크와 연결되는 장치와 연결하기 때문에 인터페이스라고 부르기도 한다.
